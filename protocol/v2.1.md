## 二、Step 2 坐标系格式（V1.0）

每个节点采用**三元锚定结构**：

```
〔节点编号〕起首词 → 终止词（含助词）
  骨架：句法主干逻辑
    ① 成分标记（藏文语法标记）= 内容
    ② 成分标记 = 内容
  标记：⚠️ 歧义预警 / 致命风险前置标注
```

**骨架标记语言（最小化）**：

| 标记 | 用途 |
|------|------|
| `因（རྐྱེན་གྱིས་）→ 果` | 因果链 |
| `条件（ན）→ 应对` | 条件句 |
| `让步（མོད་）→ 转折` | 让步句 |
| `先破（མ་ཡིན་ཏེ།）→ 立` | 是非判教句 |
| `①（དང་）②（དང་）③` | 枚举结构 |
| `⚠️ 双重否定（མིན་ཡང་）` | 须完整保留，不得软化 |
| `⚠️ 独立概念×N` | 须各别译出，不得合并 |

## 三、Step 3 对译格式（V1.0）

```
〔节点编号〕→
  汉译正文（含义补标注）
```

- 藏文不再复贴
- 翻译决策过程在内部思维链完成
- 义补分级标注沿用旧协议（A/B/C级）

---

## 四、Step 3.5 隐性执行规范

引文隔离处理强制执行以下内容（隐性）：

1. 确认引文边界（起止标记）
2. 保持字面，密义不预先解码
3. 节律注解表（逐词性质标注）
4. 解码推迟说明

---

## 五、Step 7 隐性执行规范

**静态检查项**（8项）：
1. 对译层与疏解层严格分离
2. 无后文注释误写入正文
3. 无提纲性内容误写入翻译
4. 原文顺序保持
5. 关键歧义已说明
6. Step -1 锁定术语已沿用
7. 义补分级制度已执行
8. 引文已启动 Step 3.5

**动态核查项**：本轮所有锁定术语出现次数及译法核查，异常标注。

---

## 六、Step 8 隐性执行规范

1. 完成步骤链（含触发/未触发项）
2. 本轮新增锁定术语条数
3. 未竟问题更新（继承+新增+关闭）
4. 本轮方法论备案（如有）
5. 下轮建议衔接点

---

## 七、术语管理规范

- Step -1 快照：每轮开头明文输出完整术语表
- Step 4-A 新增：每轮明文输出，自动滚动至下轮 Step -1
- 锁定后禁用译法具有约束力，违反视为致命错误
- 义补分级：A级（行文必需）可直接写入��B级须脚注标注；C级须强制删去

---

## 八、反哺校验协议（Stage 3 → Stage 2）

| 判定类型 | 处理方式 |
|----------|----------|
| ✅ 一致 | 锁定确认 |
| ⬆️ 升级 | 生成反哺修订建议，编号 R× |
| 🔴 致命错误 | 强制修订，优先级最高 |
| ⚠️ 越权 | 标注越权点，编号 W× |

反哺校验报告仍为**明文输出**，不受隐藏调整影响。
