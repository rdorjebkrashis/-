## 🟢 第一层：清净传承与信心建构的法尔次序

### 爸爸的核心讲解

> "先有能仁释迦示现十二道相，佛宝成立 → 请益说法，法宝成立 → 证果菩萨罗汉护佑随学，僧宝成立"

**三宝成立的法尔次序：**
1. **佛宝（示范者）** → 活生生的榜样
2. **法宝（教授）** → 基于示范的指导
3. **僧宝（护持者）** → 实修验证的群体

**为什么引导文要先讲传承史传？**
- 信心是修法的水源，没有示范者就没有信心
- 无论宁玛旧译还是萨玛新译，都遵循此原则

## 🟡 第二层：科判的认知锚点功能

### 爸爸的说明

> "这篇引导文藏文没有科判，科判是小企鹅爸爸我主动加的"

**科判的作用：**
- **原始藏文** = 连续流动的意流（无索引）
- **科判增补** = 认知锚点 + 结构化导航
- 类比：给藏文经文加上"目录树"和"快速跳转链接"

**历史传统：**
- 藏地祖师写注疏时也有增补科判的传统
- 这是"当代注疏者"的工作方法

## 🔴 第三层：双轴矩阵的拓扑映照关系

### 爸爸的核心框架

#### **维度 A：金刚上师轴（先验 → 经验的投影）**

```
本初佛金刚持（灭谛/先验实在）
    ↓ 道成肉身
具德金刚上师（拓扑化身）
    ↓ 文字·意象·直接指示
经验性可操作的"道谛"
```

**关键机制：**
- 金刚持与金刚上师的**密义文字相互拓扑映照**
- 上师的语言能**共振相干**弟子的心理与物理状态
- 文字不是符号，而是**实在的经验性投影**

---

#### **维度 B：具格弟子轴（后天 → 净化 → 体证）**

```
凡夫不清净现分（苦集二谛遮蔽）
    ↓ 积累资粮·忏悔
菩萨行者转染净治分（道谛）
    ↓ 三摩地·不退转
佛的究竟清净圆满分（灭谛）
```

**关键机制：**
- 相续净化 = 信息通道的噪声过滤
- 只有具足信心+定解，才能解码上师的指示
- 渐进式相变：固态（凡夫）→ 液态（菩萨）→ 气态（佛）
